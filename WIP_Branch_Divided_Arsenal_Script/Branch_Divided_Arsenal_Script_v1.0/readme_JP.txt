Branch Divided Arsenal Script by michy

1.概要
このスクリプトは、Arsenalで表示するアイテムを兵科ごとに分けるスクリプトです。このスクリプトを導入することにより、各プレイヤーごとのArsenalに表示されるアイテムが減り、準備時間を短縮することが出来るほか、各兵科の役割を明確にすることが出来ます。
スクリプトはマルチプレイヤーミッション向けに製作しており、途中参加プレイヤーにも対応しています。


2.使用方法
①スクリプトを導入したいミッションのフォルダ内に、[mission.map]フォルダ内の全てのファイルおよびフォルダをコピー&ペーストしてください。
②エディタからミッションを開き、「Arsenalを開くアクション」を適用したいオブジェクトをダブルクリックで選択し、設定画面のNAME欄に任意の名前を付け、メモを取ってください。オブジェクトは複数設定可能です。
③ミッションで使用する各ユニットのクラスネームを取得し、メモを取ってください。エディタ上では、ユニットをダブルクリックした際に表示される、設定画面の右上にクラスネームが表示されます。これらのユニットのクラスネームによって、Arsenalに表示されるアイテムが変化します。
④ミッションフォルダ内の[initPlayerLocal.sqf]ファイルを開き、構文1行目の[～forEach [a1,a2];]中の[]内に、「②でNAME欄に記述した任意の名前」を記述してください。複数ある場合は、[,]で区切ってください。
⑤ミッションフォルダ内の[scripts/BDarsenal.sqf]ファイルを開き、BDarsenal.sqf内10行目以降のホワイトリストに、Arsenalに追加したいアイテムのクラスネームを記述してください。標準では、武器(_xxWeapons)、弾薬(_Mags)、アイテム・ユニフォーム(_xxItems , _xxClothes)、バックパック(_xxBackpacks)の4カテゴリに分かれており、それぞれに兵科ごとのリストが存在します。また、ホワイトリスト欄はエディタの皆様が任意に追加することが可能です。
⑥BDarsenal.sqf内193行目以降のselect branch with classname欄に、③で決定した各兵科ごとのクラスネームと、⑥で決定したホワイトリストの中で使用するものを記述してください。
・クラスネームは、構文[～case "rhsusf_army_ocp_squadleader":～]中の[""]の中に記入してください。複数のクラスネームを適用する場合は、以下のように構文同士を[;]で区切ってください。(最終行末は必ず[:]になります)
[～	case "rhsusf_army_ocp_helipilot";	case "rhsusf_army_ocp_helicrew":]
・ホワイトリストは、構文[～[_box,(_genWeapons + _rmWeapons),false] call BIS_fnc_addVirtualWeaponCargo;～]中の[_box,]と[,false]の間に記述してください。複数のホワイトリストを使用する場合は、[()]の中に記述し、[+]で区切って下さい。
また、ホワイトリストを適用する構文は4行1セットとなっていますが、上から武器、弾薬、アイテム・ユニフォーム、バックパックとなっています。


3.サンプルミッション
このスクリプトには、サンプルミッションが付属しています。[BDA_Script_Sample.Stratis]フォルダを以下のフォルダ内に移動することで、エディタからサンプルミッションを呼び出すことが出来ます。
[C:\Users\PC username\Documents\Arma 3\missions]
もしくは
[C:\Users\PC username\Documents\Arma 3 - Other Profiles\Profile name\missions]
このスクリプトは、標準ではRHS USAF MODのユニットを使用しており、ホワイトリストにはTRI Gaming Server様で使用している各MODのアイテムが含まれています。


4.使用にあたって
本スクリプトの使用にあたって連絡等は必要ありませんが、credit欄にでも一筆入れていただければ幸いです。
また、改変・拡張もご自由に行ってください。
